Problem Name:
Selection Sort

Problem Statement:
Implement Selection Sort and print the index which gets swapped at each step.

Constraints:
1 <= t <= 100
1 <= n <= 1e3
1 <= ar[i] <= 1e9

Input Format:
The first line is the number of testcases T followed by 2T testcases. The first line is the number of elements in the array. The second line are the n space separated integers

Output Format:
Print indexes as the space separated integers. Print the output of each testcase in each line.
